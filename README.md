<h3>源码简介：</h3>
<p>国内Github下载很慢，用上了这个插件后，下载速度嗖嗖嗖的~！<p>
<p>用途：能提高中国开发者访问GitHub的速度，提升克隆Git仓库的速度，提升下载release包的下载速度。<p>
<p>用户为何安装：提高中国软件开发者的工作效率，使他们能跟专注的开发软件。<p>
<p>使用方法：<p>
<p>访问github的页面时，访问缓慢或者直接无法打开时，就直接点击插件，再次点击GitHub加速即可加速访问Github页面。<p>
